# Modelo de Mini App de conversa

Modelo responsivo de conversa programada com horários, oferta, PIX NexusPag e duas ofertas automáticas.

Antes de publicar, edite o nome e a foto em `index.html`, as mensagens e valores em `app.js` e os valores permitidos em `api/pix/create.js`.

Leia o guia completo em `..\README.md`.
